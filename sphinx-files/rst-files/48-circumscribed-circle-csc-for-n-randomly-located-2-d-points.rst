Circumscribed Circle (CSC) for N randomly located 2-D points
============================================================

**Author:** g_kalman

**Submitted on:** 2012-10-30 10:47:31-07:00

Calculates and displays Circumscribed Circle (CSC). Select N randomly located 2-D points. Display via Tkinter on Canvas N points. Find location of center (from NESW max positions). 
Radius of CSC wrt center is max distance between center and any of the random points.
